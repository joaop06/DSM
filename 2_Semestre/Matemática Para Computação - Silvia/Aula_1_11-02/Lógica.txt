- Proposição:
    Proposição é tudo aquilo que é possível realizar juízo de valor, que é possível analisar se é verdadeiro ou falso.
    Ex: 
        - É Proposição:
            s: hoje é sábado -> V
            t: 1+1 = 11 -> F

        - Não é Proposição:
            u: x+1 = 3 -> (Não é Proposição pois, por não saber o valor de x, não é possível identificar se é verdadeiro ou falso.)
            m: Choveu hoje? -> (Não é Proposição pois é uma frase interrogativa)

- Conectivos:
    - e -> (Conjunção)
    - ou -> (Disjunção)
